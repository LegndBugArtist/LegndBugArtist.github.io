info.plist 国际化

July 1, 2020 1-minute read

创建一个Strings File，并且这个文件必须命名为InfoPlist.strings

在项目中增加语言

选中 infoPlist.strings 文件，点右边的 Localize，增加你想要翻译的语言

选中 info.plst ，点击鼠标右键，选择Open As Source Code

在里面找到需要国际化的 key 的值，比如我现在设置不同语言下的 App 名称，info.plist 中的 CFBundleName 就是我需要 key

按下面两张图片的填写要翻译的内容，必须用分号结束
